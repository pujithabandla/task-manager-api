# Task Manager API

##  Overview
A RESTful backend API built using Flask that supports full CRUD operations for managing tasks.

---

##  Features
- Create tasks
- Retrieve all tasks
- Retrieve a single task
- Update tasks
- Delete tasks

---

## Tech Stack
- Python
- Flask
- REST API

---
